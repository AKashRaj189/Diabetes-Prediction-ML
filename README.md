# Early Detection of Diabetes using Machine Learning

## 📌 Project Overview
This project predicts whether a person is diabetic or not using machine learning models based on health parameters.

## 📊 Dataset
- Pima Indians Diabetes Dataset
- 768 records
- 8 input features + 1 output

## 🤖 Models Used
- Logistic Regression
- Random Forest

## ⚙️ Workflow
1. Data Preprocessing  
2. Feature Scaling  
3. Model Training  
4. Prediction  
5. Evaluation  

## 📈 Results
- Logistic Regression Accuracy: ~70%  
- Random Forest Accuracy: ~78%  

## 🧠 Conclusion
Random Forest performs better and is more suitable for diabetes prediction.
